Tag에 대하여



Tag란 무엇인가?


    태그는 HTML 문서에서 특정한 요소를 정의하고 구분하는 데 사용되는 키워드입니다. HTML은 웹 페이지의 구조와 내용을 표현하기 위한 마크업 언어이며, 태그는 이러한 문서를 어떻게 표현할지를 지정합니다.

Tag list


가장 많이 사용되는 Tag를 Chat GPT4를 이용해서 20개를 모았습니다
  1. div: 구역을 정의하는 데 사용되며, 스타일링이나 레이아웃을 위해 자주 활용됩니다.
  2. p:문단을 정의합니다.
  3. a: 하이퍼링크를 생성합니다.
  4. img: 이미지를 삽입합니다.
  5. h1 to h6: 제목을 나타내는 데 사용되며, h1이 가장 큰 제목입니다.
  6. ul: 순서 없는 목록을 정의합니다.
  7. li: 목록 항목을 정의합니다.
  8. span: 텍스트의 일부분에 스타일을 적용하거나 스크립트에서 특정 부분을 선택하는 데 사용됩니다.
  9. input: 사용자로부터 입력을 받는 데 사용되며, 다양한 종류의 입력을 처리할 수 있습니다.
  10. form: 입력 양식을 정의하며, 서버로 데이터를 전송하는 데 사용됩니다.
  11. button: 버튼을 생성합니다.
  12. strong: 텍스트를 강조하여 두껍게 표시합니다.
  13. em: 텍스트를 강조하여 이탤릭체로 표시합니다.
  14. br: 줄 바꿈을 생성합니다.
  15. hr: 수평선을 생성합니다.
  16. table: 테이블을 정의합니다.
  17. tr: 테이블의 행을 정의합니다.
  18. td: 테이블의 데이터(셀)를 정의합니다.
  19. th: 테이블의 헤더를 정의합니다.
  20. iframe: 다른 문서를 포함하는 인라인 프레임을 생성합니다.

그 외에 중요한 Tag 로는
  1. html
  2. !doctype html
  3. head
  4. body
  5. title
  6. meta

등이 있으며 이러한 Tag들은 웹 페이지의 기본 구성 요소들을 이루는데 사용됩니다.
